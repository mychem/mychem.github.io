Contributors

Contributors to Mychem are listed here:

  • Ernst-Georg Schmid - Pgchem lead developer

    Pgchem is a chemoinformatics extension for PostgreSQL developed by Ernst-Georg Schmid. We have decided that Mychem Project should have the same functions as Pgchem. Of course, we are sharing our knowledge on Open Babel, Relational Database and extension programing.

  • Chris Morley - Open Babel lead developer

    For comments and help on OpenBabel-Discuss mailing list.

  • Noel O’Boyle - Open Babel developer

    For comments and help on OpenBabel-Discuss mailing list.

  • Hans de Winter

    For debugging and tests on Mac OS X.
